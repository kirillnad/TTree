# Пользовательская шпаргалка

## Допустимые к загрузке файлы
- Изображения: любые `image/*` (JPEG/PNG/GIF/WebP и т.п.).
- Документы и таблицы: `pdf`, `doc`, `docx`, `xls`, `xlsx`, `ppt`, `pptx`.
- Текстовые: `txt`, `csv`, `rtf`.

Файлы сохраняются в `/uploads/{articleId}/uuid.ext`. В блок автоматически вставляется ссылка с оригинальным именем файла; на диске имя уникализируется.

## Горячие клавиши

### Просмотр (режим view)
- `Enter` — перейти к редактированию выбранного блока.
- `Ctrl+Z` / `Ctrl+Y` — шаг назад / вперёд.
- `Ctrl+Shift+↑ / Ctrl+Shift+↓` — переместить блок вверх/вниз.
- `Ctrl+↑ / Ctrl+↓` — создать новый блок до/после текущего.
- `Ctrl+Delete` — удалить текущий блок.
- `Ctrl+→ / Ctrl+←` — увеличить/уменьшить вложенность блока.
- `↑ / ↓` — выбрать предыдущий/следующий блок.
- `←` — свернуть текущую ветку (если можно).
- `→` — развернуть текущую ветку (если можно).

### Редактирование блока (режим edit)
- `Ctrl+Enter` — сохранить блок.
- `Ctrl+→ / Ctrl+←` — изменить вложенность, оставаясь в редактировании.
- `Esc` — отменить редактирование блока.

### Редактирование заголовка статьи
- `Enter` — сохранить заголовок.
- `Esc` — отменить редактирование заголовка.
